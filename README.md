# ShellCourse

Creating env variables:

1. Enter into /etc/profile and edit using sudo vim.
2. Insert a var using 
```
#Var
MY_ENV_VAR=Value of the env.
export MY_ENV_VAR
```
3. Save the file.
4. To get a variable we should follow the syntax.
```
#Var into program
edad=29
name=Omar
echo "Nombre: $nombre"
```
